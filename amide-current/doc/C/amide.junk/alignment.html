Aligning Data Sets

AMIDE currently supports two methods for aligning data sets to each other. They are described below.

Manual Alignment

Manual Alignment

  1. First rotate the data sets so that they are level with respect to the transverse, coronal, and sagittal views.

  2. Set the center of each of the data sets to the origin (0,0,0).

  3. Choose one of the data sets to be the "fixed" data set.

  4. Then slowly alter the center of the data sets relative to the "fixed" data set until the data sets overly each other as desired.
