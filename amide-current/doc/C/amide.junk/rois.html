ROI stands for Region of Interest. ROI's designate a volume on the object over which statistics should be calculated.

ROI Types

The following ROI types are currently supported.

Ellipsoid

An ellipsoid is similar to a sphere, but with a radius specified for each direction [x,y,z]. In the case of x=y=z, the ellipsoid is a sphere.

Elliptic Cylinder

An elliptic cylinder is similar to a regular cylinder, except it has an ellipse as it's base instead of a circle.

Box

Exactly what it says, a 3D box.

Group

Not currently implemented. This will someday allow ROI's to be defined in groups, so that ROI's can be manipulated both in groups and individually.
