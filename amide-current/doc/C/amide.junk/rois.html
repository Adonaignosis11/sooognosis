Chapter 2. Using ROI's

Using ROI's

ROI stands for Region of Interest. ROI's designate a volume on the object over which statistics should be calculated.

ROI Types

The following ROI types are currently supported.

Ellipsoid

An ellipsoid is similar to a sphere, but with a radius specified for each direction [x,y,z]. In the case of x=y=z, the ellipsoid is a sphere.

Elliptic Cylinder

An elliptic cylinder is similar to a regular cylinder, except it has an ellipse as it's base instead of a circle.

Box

Exactly what it says, a 3D box.

Group

Not currently implemented. This will someday allow ROI's to be defined in groups, so that ROI's can be manipulated both in groups and individually.

Drawing ROI's

To draw an roi, first select the ROI desired under the "add roi:" pull down menu. Amide will request a name for this ROI and add it to the study item list, but no roi will be drawn. To draw this new roi, you need to first double click on the roi in the study item list, and then draw the ROI on the desired view. The x and y dimensions of the ROI will be as drawn. The z dimension ("depth") of the newly drawn ROI will be the same as the current image thickness.

Manipulating ROI's

After an ROI is drawn, it can be further manipulated to adjust it's size, placement, and orientation. First double click the ROI in the study item list to ensure that this ROI is the currently selected item. You can then directly manipulate the ROI by clicking on it in any of the volume view windows. Mouse button 1 is used for zooming the ROI. Mouse button 2 is used to rotate the ROI. And Mouse button 3 is used to shift the ROI. You can also edit the ROI size/placement/orientation/name etc. by clicking on mouse button 3 while over the ROI's name in the study item list. This brings up the edit ROI dialog, which allows keyboard entry of the ROI's parameters.
