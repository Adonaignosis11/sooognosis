ROI stands for Region of Interest. ROI's designate a volume on the object over which statistics should be calculated.

ROI Types

The following ROI types are currently supported.

Ellipsoid

An ellipsoid is similar to a sphere, but with a radius specified for each direction [x,y,z]. In the case of x=y=z, the ellipsoid is a sphere.

Elliptic Cylinder

An elliptic cylinder is similar to a regular cylinder, except it has an ellipse as it's base instead of a circle.

Box

Exactly what it says, a 3D box.

Isocontours

Isocontour ROI's are regions selected from the data set such that the edge values of the ROI are always the same value. There are two types, 2D and 3D isocontours.

2D Isocontour

A 2D isocontour is derived by considering a value on one of the displayed 2D slices. The depth of a 2D isocontour is specified initially by the depth of the viewed slices.

3D Isocontour

A 3D isocontour is derived by considering a value on the current frame of the active data set.
