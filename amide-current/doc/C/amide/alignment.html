Aligning Data Sets

AMIDE currently supports two methods for aligning data sets to each other. They are described below.

Manual Alignment

Manual Alignment

  1. First rotate each of the data sets so that they are level with respect to the transverse, coronal, and sagittal views. This is easily done using the shift-2, and shift-3 mouse combinations, but can also be done from the data set modification dialog.

  2. Choose one of the data sets to be the active data set. The other data set will be the "fixed" data set.

  3. Shift the active data set so that the two data sets line up appropriately. This is easily done using the shift-1 mouse combination, but can also be done from the data set modification dialog.

  4. If fine tuning adjustments are needed, these are best done from the data set modification dialog.
