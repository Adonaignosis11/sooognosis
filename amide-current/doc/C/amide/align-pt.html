Drawing Alignment Points

Alignment points can be added to any of the data sets in the study. To add an alignment point, hit the "Edit->Add alignment point" menu item, which will add an alignment point to the currently active data set. Alternatively, hold down the shift button and press the right mouse button while hovering over a data set's name in the study list. In either case, you will then be prompted for a name for the alignment point, and the point will then be placed at the currently viewed location. The alignment point can be moved by clicking on the marker point shown in any of the views.

Alignment Point Modification Dialog Box

To modify parameters of an alignment point, first select and then right click the point in the study list. The Alignment Point Modification Dialog Box will then appear. Parameters that can be modified are listed below.
