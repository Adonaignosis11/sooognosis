Drawing ROI's

To draw an ROI, first select the ROI desired under the "add ROI:" pull down menu. Amide will request a name for this ROI and add it to the study item list, but no ROI will be drawn. To draw this new ROI, you need to first make the new ROI the active object by either double clicking with the left button or clicking with the middle button the ROI in the study item list. After selecting the ROI, it can be drawn on any of the displayed views of the current volumes. The x and y dimensions of the ROI are controlled by the size of the drawing. The z dimension ("depth") of the newly drawn ROI is determined by the current image thickness.
