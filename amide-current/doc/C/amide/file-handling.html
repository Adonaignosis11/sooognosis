AMIDE saves studies in an extensible XML based directory format called XIF (Xml Image Format). These data directories characteristically end with ".xif", and appear as files within AMIDE.

Format of Saved Studies

Although admittedly annoying from a data transfer standpoint, using a directory structure for saving study information has the decisive advantage of making the saved information easily excessible using standard command-line utilities and text-based tools.

Each XIF directory contains a file called "Study.xml" which contains the basic study parameters. Additional files can also be found in the XIF directory, such as ROI_*.xml files which contain ROI's, and Volume_*.xml files and their corresponding Volume_*.dat files, which contain the image data set parameters and the raw data respectively. The raw data file format is abitrary, and is little endian 32-bit float by default.
