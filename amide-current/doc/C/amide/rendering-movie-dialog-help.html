Rendering Movie Dialog

Movie Parameters

Frames

How many frames should be in the MPEG1 movie. The MPEG1 movies generated will be set to run at 30 frames/second, so the default of 300 frames will give a ten second movie.

Rotations on [x,y,z]

This setting determines how many times the data set will be rotated around the given axis over the course of the movie. Note that the rotations will be done around the initial axis (frame 0), not the axis of the previous frame. Additionally, the rotation for each frame is done in x->y->z order (rotate on x first, then y, then z).
