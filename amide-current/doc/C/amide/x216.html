Drawing ROI's

To draw an ROI, first select the ROI desired under the "add ROI:" pull down menu. Amide will request a name for this ROI and add it to the study item list, but no ROI will be drawn. The next mouse input on the displayed views will be used to draw the new ROI. For ellipsoid, elliptic cylinder, and box ROI's, The x and y dimensions of the ROI are controlled by the size of the drawing. The z dimension ("depth") of the newly drawn ROI is determined by the current image thickness. For isocontour's, the value of the data set at the clicked upon location will be used to derive the isocontour.
